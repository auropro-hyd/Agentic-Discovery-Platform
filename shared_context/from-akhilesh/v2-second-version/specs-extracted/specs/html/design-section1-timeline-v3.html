Design — Revised O2C Demo: 20-Minute Structure

Discovery is the setup. The output suite is the show. Back half of the demo is entirely output — three deliverables, one after the other.

Setup + Discovery: 0–8 min
Output Suite: 8–18:30 min
The Offer: 18:30–20 min
— Setup + Discovery —
0:00 – 1:30
The Setup (1.5 min) — Name the business identity gap. Show documents loaded. One-line on how it works: "It reads everything at once, maps what's connected, and surfaces what's broken." No platform mechanics. No architecture.
1:30 – 5:00
Discovery Runs (3.5 min) — Compressed. Platform reads inputs, findings appear. Narration stays business-level. Add one explicit line: "We intentionally left some gaps in these synthetic documents — you'll see the platform catch them." 3 findings surface. Pause only on Finding 2 (67% undocumented EDI). Others land fast.
5:00 – 8:00
Gaps Resolved (3 min) — Finding 1: one targeted call, three ranked options, resolution selected. Finding 2 flagged for documentation sprint. Finding 3 goes to Digital + Legal. Narration: "Not a workshop. A targeted call. The platform already ranked the options — you confirm."
— Output Suite — the deliverable —
8:00 – 11:00
Current State Assessment (3 min)
Three screens, one minute each: (1) Domain map — two O2C flow paths, documented vs. actual. (2) Context map — how O2C connects to Demand Planning and where the cross-domain failure pattern lives. (3) Process diagram — the EDI flow as it actually runs, with ownership gaps highlighted. Narration: "This is your O2C landscape as it actually operates."
11:00 – 16:30
✦ AI Opportunity Portfolio (5.5 min) — PRIMARY OUTPUT Main deliverable
3 opportunity cards. Each card: opportunity name, intervention type badge (AI Agent / Automation Pipeline / HITL Workflow), business impact figure derived from inputs, source documents named. ~1.5–2 min per card. Narration: "The platform doesn't just find the problems. It tells you what type of intervention each one requires and what it costs you to leave it alone."
Opp 1: HITL — Customer Master Reconciliation Opp 2: Automation — EDI Exception Pipeline Opp 3: AI Agent — Credit Decisioning
16:30 – 17:30
Strategic Roadmap (1 min) — SECONDARY Complements portfolio
One screen. Now / Next / Later. The three portfolio opportunities mapped to a sequence. Narration: "We've also mapped how these sequence. The customer master fix is the foundation — everything else builds on top of it." 60 seconds. No line-by-line narration.
— The Offer —
17:30 – 20:00
The Offer (2.5 min) — Hard close. Portfolio already landed. Script unchanged in substance. Close signal: "Can we run this on our real O2C landscape?" Hand floor back. Silence is fine.
Setup + Discovery
8 min · compressed · business language only
Output Suite
10.5 min · Current State + Portfolio + Roadmap
Offer
2.5 min · no qualifications · silence is fine
